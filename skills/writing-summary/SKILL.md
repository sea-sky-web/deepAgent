# writing-summary

## Overview

This skill helps convert raw research notes into a clear technical explanation.

## Instructions

1. Start from the core conclusion.
2. Keep the structure compact and avoid unnecessary repetition.
3. Prefer "what / why / how / trade-off" when summarizing technical material.
4. If the source material is fragmented, merge overlapping points before writing.
